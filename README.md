## 프로토타입

- [Unity 탐색 박스 프로토타입](game/README.md) — 확장 맵, 실제 벽, 수리·배신·추격 봇 3명 포함
- Unity `6000.5.3f1`에서 `game` 폴더를 열고 빈 Scene에서 Play하면 테스트 공간이 자동 생성됩니다.

현재 프로토타입은 핵심 조작의 손맛을 확인하는 오프라인 v0.2 검증 빌드입니다. 온라인 멀티플레이, 음성 채팅, 메타 보상, 맵 변형과 최종 그래픽은 아직 구현하지 않았습니다.

## 현재 방향

- Unity 6 LTS 기반 스타일라이즈드 3D
- 친구 4명, 스테이지당 5~7분, 총 3스테이지
- 고정 임포스터 없이 플레이 중 자발적으로 발생하는 배신
- 근접 음성, 협동 수리, 사원증 위조, 물리적 마지막 버튼
- 2인 팀과 AI 보조를 전제로 한 6~12개월 초기 출시 계획
- 출시 이후 확장 가능한 맵, 몬스터, 스킨, 게임 타입

기획은 반복 검증 중이며 현재 최신안은 v0.12입니다. Unity 프로토타입에는 검증이 끝난 규칙만 이후 반영합니다.
