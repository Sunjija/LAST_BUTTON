# LAST BUTTON

고장 난 우주선을 함께 수리하면서, 누구든 비밀 계약을 받아들이고 팀의 전력을 훔쳐 혼자 탈출할 수 있는 4–6인 멀티플레이 사회적 딜레마 게임 기획 프로젝트입니다.

## 프로토타입

- [Unity 회색 박스 프로토타입](game/README.md) — 확장 맵, 실제 넉백, 수리·배신·추격 봇 3명 포함
- Unity `6000.5.3f1`에서 `game` 폴더를 열고 빈 Scene에서 Play하면 테스트 공간이 자동 생성됩니다.

## 현재 구현 상태

- 공동 수리와 공동 탈출
- 키카드 탈취, 공동 전력 전용, 개인 포드 충전과 단독 탈출
- 빗나갈 수 있는 돌진형 밀치기와 키카드 드롭
- 배신자용 질주와 일회용 보안 교란기
- 수리·추격·배신을 수행하는 봇 3명
- 격벽, 문틈, 우회로가 있는 테스트 맵과 봇 경로 탐색
- 6분 제한 시간, 결과 화면, 개발용 보상 및 사건 HUD
- Unity 배치 검증과 Windows 개발 빌드

현재 빌드는 핵심 규칙의 작동 여부를 확인하는 오프라인 프로토타입입니다. 온라인 멀티플레이, 음성 채팅, 메타 보상, 맵 변형과 최종 그래픽은 아직 구현하지 않았습니다.

## 현재 방향

- Unity 6 LTS 기반 스타일라이즈드 3D
- 스테이지당 5–7분, 3스테이지 세션
- 고정 임포스터 없이 플레이 중 자발적으로 발생하는 배신
- 근접 음성, 협동 수리, 물리적 버튼 저지
- 2인 팀과 AI 보조를 전제로 한 60일 초기 출시 계획
- 출시 이후 회피·유인·격리·역이용이 가능한 몬스터 확장

기획의 현재 기준은 v0.5입니다. 현재 Unity 프로토타입은 v0.2 규칙을 검증한 빌드이므로 개인 순위, 합법 업무, 순차 블랙 계약, 이중 용도 협동 작업, 출입 권한 정지와 3스테이지 보상 경제는 아직 반영되지 않았습니다.
