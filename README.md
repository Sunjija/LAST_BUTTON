# LAST BUTTON

고장 난 우주선을 함께 수리하면서, 누구든 비밀 계약을 받아들이고 팀의 전력을 훔쳐 혼자 탈출할 수 있는 4~6인 멀티플레이 사회적 딜레마 게임 기획 프로젝트입니다.

## 현재 방향

- Unity 6 LTS 기반 스타일라이즈드 3D
- 스테이지당 5~7분, 3스테이지 세션
- 고정 임포스터 없이 플레이 중 자발적으로 발생하는 배신
- 근접 음성, 협동 수리, 물리적 버튼 저지
- 2인 팀과 AI 보조를 전제로 한 60일 초기 출시 계획
