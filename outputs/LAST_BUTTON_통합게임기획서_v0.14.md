# LAST BUTTON 통합 게임 기획서 v0.14

작성일: 2026-07-12  
문서 상태: 차별성 재설계 2차안  
대상: 마이크를 사용하는 친구 4명  
플레이 시간: 스테이지당 5~7분, 총 3스테이지

## 1. 게임 정체성

네 명이 우주선을 수리하다 황금 배터리 소지자가 개인 탈출정으로 달아나면, 추격자가 그를 밀쳐 배터리를 빼앗고 `팀에 반환할지 자신이 이어서 탈출할지` 몸의 방향으로 선택하는 협동·욕심 전염 액션 게임.

```text
배신자를 찾는 게임 X
배신할 수 있는 물건이 사람 사이를 옮겨 다니는 게임 O
```

회의, 투표, 처형, 고정 임포스터, 후보 압축, 직업 추리는 없다. 배신 실행 전에는 모두 협동하고 실행 순간부터 소지자가 공개된다.

## 2. 3초 차별성 테스트

```text
도망자를 돌진으로 밀침
→ 황금 배터리가 튕겨 나옴
→ 추격자가 주움
→ “원자로 반환 1.5B / 이 배터리 현상금 2~3B” 표시
→ 추격자가 원자로 대신 탈출정으로 방향을 틂
```

무음 영상만 보고 `잡던 사람이 새 도망자가 됐다`가 이해되어야 한다.

## 3. 개인 결과와 성장

세션 동안 모은 지분으로 급여 등급을 정한다.

- 0~1.99B: C급
- 2~2.99B: B급
- 3~3.99B: A급
- 4B 이상: S급

단일 1등이나 탈락은 없다. 공동 발진은 `+1B`, 수리 개인 성과는 스테이지당 최대 `+0.5B`다. 충성 플레이만으로도 S급에 도달할 수 있다.

누적 지분 최고 기록이 처음 정수 단위를 넘을 때마다 일회용 보급 카드 한 장을 받는다. 다음 스테이지에 한 장만 장착하며 사용해도 최종 지분은 줄지 않는다.

- 충격 흡수화: 밀치기 또는 미끄러짐 1회 감소
- 예비 퓨즈: 공동 설비를 4초 임시 가동
- 문 우회키: 잠긴 문 하나를 짧게 개방
- 호스 클램프: 냉각수 사고 1회 봉합

## 4. 한 스테이지 흐름

1. 고장 설비·통로·퓨즈함·탈출정 후보 공개
2. 3~4분 협동 수리와 에너지 셀 운반
3. 중간에 비공개 블랙 계약 제안
4. 계약자는 예비 펄스 3개를 모음
5. 네 명이 3개 발진 설비에 배치
6. 계약자가 원격 퓨즈함에서 황금 배터리 가동
7. 소지자 공개, 퓨즈함 반대편 탈출정 개방
8. 설비 유지와 추격, 배터리 손바뀜
9. 마지막 버튼 또는 공동·비상 발진
10. 즉시 지분 정산

## 5. 협동 수리와 코미디

- 전력 릴레이: 떨어진 두 방에서 박자를 맞춰 입력
- 냉각수 호스: 호스를 끌고 밸브를 조작, 꼬이면 물줄기와 미끄러운 바닥 발생
- 압력 밸브: 숫자를 전달하고 과압을 조절, 실패하면 짧은 증기 분출
- 에너지 셀: 무겁고 흔들리는 셀을 함께 운반

실패는 10~20초 안에 복구되며 전체 진행도를 초기화하지 않는다. 개인 성과는 도움·정확도·운반량에서 얻고 한 업무 반복으로 상한을 채울 수 없다.

## 6. 예비 펄스

에너지 셀 연결 때 운반자는 한 번 선택한다.

- 전량 공급: 공동 발진 안정도와 개인 수리 성과 증가
- 펄스 추출: 안정도와 수리 성과를 포기하고 일회용 개인 도구 충전

비계약자는 펄스를 최대 2개 저장할 수 있다. 사용하지 않은 펄스는 발진 준비 때 자동으로 공동 안정도로 반환된다. 따라서 필요할 때까지 보험처럼 들고 있을 수 있지만 도구로 소비하면 팀의 안전 여유가 실제로 줄어든다.

계약자는 세 번째 펄스를 숨겨진 계약 배터리에 넣을 수 있다. 세 번째 추출은 합법 용량을 넘지만 다른 플레이어가 현장을 직접 보지 않으면 사람을 특정하는 로그나 회의는 발생하지 않는다.

## 7. 블랙 계약과 보증금

- 무작위 한 명에게 12초 비공개 제안
- 거절하면 다른 한 명에게 마지막 제안
- 계약금 `2.5B / 3B / 3.5B` 중 무작위
- 한 세션에서 계약 수락 또는 단독 탈출 성공은 플레이어당 1회
- 계약 존재와 수락자는 실행 전까지 비공개

수락 즉시 `0.25B`의 보증금이 잠긴다.

- 실행하지 않고 포기: 보증금 `-0.25B`, 저장 펄스 공동 반환
- 배터리 가동 후 실패: 총 `-0.5B`
- 성공: 보증금 반환 후 계약금 전액 지급

수락은 무료 옵션이 아니다. 계약 화면에는 금액, 손실, 현재 지분과 성공 시 도달 등급을 표시한다.

## 8. 정상 발진의 인원 딜레마

원자로·냉각수·통신 세 설비를 각각 한 명이 잡고 8초 동안 `3-of-3`으로 유지해야 공동 발진한다. 네 번째 플레이어만 사고 복구와 몬스터 유인을 담당하는 자유 인원이다.

계약 경보가 울리면 다음 선택이 생긴다.

- 자유 인원 한 명만 추격: 발진은 유지되지만 계약자 저지가 어려움
- 설비 담당까지 추격: 추격자는 늘지만 비어 있는 설비 게이지가 감소
- 전원 추격: 배터리를 잡기 쉽지만 공동 발진은 완전히 중단

누가 남고 누가 쫓을지를 음성으로 즉시 정해야 한다.

## 9. 배터리 가동과 최초 계약자의 우위

계약자는 펄스 3개를 모은 뒤 원격 암시장 퓨즈함에서 배터리를 가동한다. 가동 즉시 소지자와 탈출정 위치가 전원에게 공개된다.

최초 계약자에게만 다음 실행 우위가 있다.

- 퓨즈함과 탈출정의 정확한 조합을 실행 전부터 앎
- 가동 직후 2초 가속
- 원하는 문 하나를 4초 잠그는 일회용 교란
- 성공 시 원래 계약금 수령

퓨즈함과 탈출정은 맵 반대 구역의 후보끼리 연결한다. 계약자는 준비 비용을 내는 대신 경로와 첫 움직임을 통제한다.

## 10. 황금 배터리의 물리 규칙

- 방향 고정 돌진에 맞으면 떨어짐
- 벽과 물체에 튕겨 굴러감
- 누구나 주울 수 있음
- 자발적으로 버리면 5초 뒤 원자로 반환구 앞으로 이동
- 바닥에 있는 동안 누구든 다시 주울 수 있음
- 마지막 버튼을 5초 유지해야 단독 탈출

돌진은 조준 방향이 고정되고 빗나가면 짧은 후딜이 있다. 배터리 소유권은 마지막 버튼이 완료될 때까지 여러 번 바뀔 수 있다.

## 11. 욕심의 전염과 현상금 승계

비계약자가 배터리를 주우면 배터리에 남은 현상금이 표시된다.

- 2.5B 계약 배터리 → 기회 탈출 `2B`
- 3B 계약 배터리 → 기회 탈출 `2.5B`
- 3.5B 계약 배터리 → 기회 탈출 `3B`

새 소지자는 최초 계약자의 가속과 문 잠금을 받지 않는다. 준비 비용은 없지만 주변 추격자 사이에서 출발하고 현상금도 `0.5B` 적다.

선택은 UI 버튼이 아니라 이동으로 한다.

- 원자로 반환구에 넣고 공동 발진 성공: 기본 `+1B`와 반환 `+0.5B`
- 탈출정 방향으로 3초 연속 이동: 새 도망자로 공개, 현상금 승계

다른 사람이 새 도망자로 확정되는 순간 이전 시도자는 실패 `-0.5B`가 확정된다. 새 도망자도 저지되거나 반환되면 `-0.5B`다.

## 12. 종료 분기

### 배터리 반환

- 3-of-3 공동 발진 재개
- 반환자: 공동 성공 시 총 `+1.5B`
- 나머지 충성 플레이어: `+1B`
- 실패한 도망 시도자: `-0.5B`

### 단독 탈출

- 최초 계약자는 계약금, 기회 탈출자는 승계 현상금 획득
- 남은 세 명은 20초 동안 `2-of-3` 비상 발진
- 비상 발진 성공: 각각 `+0.5B`
- 실패: 남은 플레이어 보상 없음

### 전체 시간 초과

- 전원 `-0.25B`
- 현재 도망 시도자는 추가 손실을 포함해 총 `-0.75B`

개인 탈출은 나머지를 즉시 제거하지 않고 더 어려운 마무리를 남긴다.

## 13. 계약이 없는 스테이지

아무도 계약을 수락하지 않으면 발진 준비 중 공개 `성과 캡슐`이 자유 인원에게 지급된다.

- 공동 화물칸에 넣고 발진: 전원 개인 성과 `+0.1B`
- 개인 보관함에 5초간 넣기: 소지자만 `+0.4B`, 공동 안정도 감소
- 캡슐은 밀치면 떨어지고 다른 사람이 가져갈 수 있음
- 개인 보관은 단독 탈출이나 팀 전멸을 만들지 않음

계약 없는 판에도 작은 욕심의 손바뀜은 남지만 황금 배터리와 마지막 버튼의 보상을 대체하지 않는다.

## 14. 충성 플레이어의 능동적 재미

- 전량 공급과 펄스 보험 선택
- 세 설비의 역할 교대와 자유 인원 운영
- 수리 사고 복구와 몬스터 유인
- 설비를 버리고 추격할지 결정
- 조준 돌진으로 배터리 탈취
- 획득 즉시 반환과 현상금 승계 중 선택
- 단독 탈출 뒤 세 명의 비상 발진

계약 제안을 받지 못해도 황금 배터리를 잡는 순간 같은 핵심 선택에 참여한다.

## 15. 맵과 몬스터

고정 방 연결 골격에서 고장 설비, 잠긴 문, 퓨즈함, 탈출정, 느린 순찰 몬스터의 위치와 소음 반응을 바꾼다.

몬스터는 운반자와 큰 소음을 쫓고 문·전력·수리 동선을 방해한다. 배터리 소지자는 몬스터를 추격자에게 유인할 수 있고 충성자는 반대로 길을 열 수 있다. 전투와 영구 사망은 없다.

## 16. DDD 리듬

```text
협동 수리의 작은 성공
→ 물리 사고와 복구
→ 지분·펄스·다음 장비 진행
→ 손실이 있는 비밀 계약
→ 세 번째 펄스의 관찰 위험
→ 공개 황금 배터리 경보
→ 설비 유지와 추격 인원 협상
→ 빗나갈 수 있는 돌진과 배터리 드롭
→ 추격자에게 현상금 유혹 전염
→ 마지막 버튼 5초 니어미스
→ 단독 탈출과 3인 비상 발진 또는 공동 성공
→ 즉시 정산
```

## 17. 차별성 금지선

- 회의·처형 투표·후보 압축을 넣지 않음
- 숨은 직업 조합 암기가 핵심이 되지 않음
- 별도 증거 수집 단계가 물리 행동보다 길지 않음
- 배신자를 관전자로 제거하지 않음
- 마지막 버튼보다 정체 추리가 중요해지지 않음
- 황금 배터리를 주운 충성자가 새 배신자가 될 수 있어야 함

## 18. 초기 검증 범위

- 친구 4명, Discord 음성
- 맵 1개, 위치 조합 3개
- 수리 4종, 코미디 사고 2종
- 펄스와 개인 도구 2종
- 계약금 3단계와 보증금
- 퓨즈함·탈출정 각 2곳
- 황금 배터리 물리, 반환구, 현상금 승계
- 3-of-3 정상 발진, 2-of-3 비상 발진
- 방향 고정 돌진과 마지막 버튼
- 계약 없는 판의 성과 캡슐

작업 도장, 후보 패널, 보정 키, 회의, 투표, 자체 음성, 공개 매칭과 영구 성장은 제외한다.

## 19. 검증 기준

- 10초 무음 영상에서 배터리 탈취 후 새 배신 가능 이해율 `70% 이상`
- 게임 설명 후 어몽어스형 추리 게임으로 분류하는 비율 `25% 미만`
- 계약 수락률 `25~55%`
- 계약 포기율 `10~35%`
- 최초 계약자 성공률 `25~45%`
- 배터리 소유권 변경 평균 `0.8~2회`
- 비계약자의 획득 후 기회 탈출 선택률 `20~45%`
- 반환과 기회 탈출을 실제로 고민했다는 응답 `60% 이상`
- 경보 후 설비 담당이 추격으로 이탈한 스테이지 `30~70%`
- 계약 없는 스테이지 재미 응답 `60% 이상`
- 충성·계약 만족도 차이 1점 이내
- 마지막 버튼 니어미스 `30% 이상`
- 세 스테이지 재플레이 의향 `60% 이상`

## 20. 폐기 기준

12회의 4인 세션 뒤 다음 중 하나가 지속되면 해당 구조를 폐기한다.

- 배터리 획득자가 거의 항상 반환하거나 거의 항상 탈출
- 자유 인원만 추격하는 것이 항상 정답
- 설비 담당 한 명이 반복적으로 재미를 희생
- 펄스 추출이 항상 최대 또는 항상 0으로 고정
- 계약 없는 판이 성과 캡슐을 포함해 수리 노동으로 느껴짐
- 원계약보다 남의 배터리를 노리는 전략이 우세
- 배신이 욕심 선택이 아니라 트롤로 받아들여짐
- 플레이어가 핵심을 배신자 추리로 설명함

## 21. 현재 상태
