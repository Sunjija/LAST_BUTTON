# LAST BUTTON 통합 게임 기획서 v0.16

작성일: 2026-07-12  
문서 상태: 차별성 반복 검증 기준안  
대상: 마이크를 사용하는 친구 4명  
플레이 시간: 스테이지당 4~6분, 총 4스테이지

## 1. 한 줄 콘셉트

네 명이 공동 발진에 필요한 황금 배터리를 한 번씩 운반하며, 원자로에 꽂으면 모두 보상받지만 개인 탈출정으로 방향을 틀면 혼자 큰돈을 노릴 수 있는 협동·욕심 전염 액션 게임.

도망자를 밀쳐 배터리를 빼앗은 추격자도 반환하거나 새 도망자가 될 수 있다.

## 2. 장르와 금지선

```text
배신자를 찾는 게임 X
공동 물건을 든 사람이 어디로 가는지 지켜보는 게임 O
```

- 고정 임포스터 없음
- 회의, 투표, 처형, 후보 압축 없음
- 숨은 역할을 알아맞혀야 하는 진행 없음
- 황금 배터리는 매 스테이지 등장
- 배신 선언 즉시 소지자 공개
- 배터리 소유권과 배신 기회가 함께 이전
- 마지막 버튼 앞 물리 행동이 클라이맥스

작업 도장, 보정 키와 두 후보 패널은 폐기한다.

## 3. 대표 장면

```text
세 명이 발진 설비를 준비하고 한 명이 황금 배터리를 운반
→ 운반자가 갈림길에서 탈출정 쪽으로 방향을 틂
→ 경보와 함께 추격 시작
→ 돌진에 맞아 배터리가 튕김
→ 배터리를 주운 추격자가 원자로 대신 탈출정으로 달림
→ 마지막 버튼을 두고 소유권이 다시 바뀜
```

10초 무음 영상에서 `잡던 사람이 새 도망자가 됐다`가 이해되어야 한다.

## 4. 세션 구조와 공정한 운반 순환

한 세션은 4스테이지며 각 플레이어가 황금 배터리 최초 운반자를 정확히 한 번 맡는다.

- 첫 스테이지 시작 때 무작위 운반 순서 공개
- 스테이지마다 다음 사람으로 자동 교대
- 최초 운반 역할은 협상이나 투표로 바꿀 수 없음
- 다른 플레이어는 실제 도주 선언 또는 블랙 계약 강탈 뒤에만 배터리를 넘겨받을 수 있음

모든 플레이어가 한 번은 핵심 방향 선택을 경험한다. 총 플레이 시간은 16~24분이다.

## 5. 개인 결과

4스테이지 동안 개인 지분 `B`를 모은다.

- 0~1.99B: C급
- 2~3.49B: B급
- 3.5~4.99B: A급
- 5B 이상: S급

단일 1등과 탈락은 없다.

- 공동 발진 성공: 전원 `+1B`
- 수리 개인 성과: 스테이지당 최대 `+0.5B`
- 기본 단독 탈출: `+2B`
- 블랙 계약 단독 탈출: `+2.5B / +3B / +3.5B`
- 도주 선언 또는 강탈 뒤 실패: `-0.5B`

공동 성공 4회와 평균 개인 성과 `0.25B` 이상으로 충성 플레이도 S급에 도달한다.

한 플레이어는 세션당 단독 탈출 보상을 한 번만 받을 수 있다. 이미 성공한 플레이어가 이후 배터리를 들면 개인 탈출정이 잠기고 공동 소켓 반환만 가능하다.

## 6. 다음 스테이지 보급

누적 지분 최고 기록이 처음 정수 단위를 넘을 때마다 일회용 보급 카드 한 장을 얻는다. 다음 스테이지에 한 장만 장착하며 사용해도 최종 지분은 줄지 않는다.

- 충격 흡수화: 밀치기 또는 미끄러짐 1회 감소
- 예비 퓨즈: 설비를 4초 임시 가동
- 문 우회키: 잠긴 문 하나를 짧게 개방
- 호스 클램프: 냉각수 사고 1회 봉합

## 7. 한 스테이지 흐름

1. 고장 설비·통로·현재 운반자 공개
2. 3분 내외 협동 수리와 물리 사고
3. 중간에 비공개 블랙 계약 제안
4. 수리 완료 후 세 명은 발진 설비, 지정 운반자는 배터리 제작기로 이동
5. 역할 위치를 3초 유지하면 발진 단계 잠금
6. 운반자는 공동 소켓 또는 탈출정 방향 선택
7. 도주 또는 계약 강탈 시 공개 추격과 배터리 손바뀜
8. 공동 발진 또는 마지막 버튼과 3인 비상 발진
9. 즉시 정산

## 8. 협동 수리

- 전력 릴레이: 떨어진 두 방에서 박자를 맞춤
- 냉각수 호스: 호스를 끌고 밸브를 조작, 꼬이면 물과 미끄러운 바닥 발생
- 압력 밸브: 숫자를 전달하고 과압을 조절, 실패하면 짧은 증기 분출
- 에너지 셀: 무거운 셀을 함께 운반

실패는 10~20초 안에 복구되고 전체 진행도를 초기화하지 않는다. 개인 성과는 도움·정확도·운반량으로 얻으며 한 업무 반복으로 상한을 채울 수 없다.

## 9. 발진 역할 잠금

수리가 끝나면 지정 운반자에게 배터리 제작기 위치가 표시되고 나머지 세 명에게 원자로·냉각수·통신 설비가 하나씩 배정된다.

네 명이 각 위치를 3초 유지하면 역할이 잠긴다.

- 지정 운반자는 황금 배터리와 자기 잠금 결합
- 도주 선언 전 일반 밀치기로는 배터리가 떨어지지 않음
- 플레이어 몸은 여전히 밀려 경로 방해와 코미디는 발생
- 세 설비는 배터리가 공동 소켓에 들어온 뒤 8초간 3-of-3 유지
- 설비 담당이 자리를 뜨면 해당 게이지 감소

이 잠금은 배신 전 무의미한 배터리 스폰 난투를 막는다.

## 10. 블랙 계약

- 무작위 한 명에게 12초 비공개 제안
- 거절하면 다른 한 명에게 마지막 제안
- 계약금 `2.5B / 3B / 3.5B` 중 무작위
- 수락 즉시 보증금 `0.25B` 잠금
- 계약 수락과 단독 탈출 성공은 각각 세션당 1회
- 계약 존재와 수락자는 실행 전까지 비공개

수락자는 `검은 우회권`을 얻는다.

- 제작기와 활성 탈출정 조합을 미리 확인
- 자신이 지정 운반자면 도주 선언 직후 2초 가속과 4초 문 잠금
- 지정 운반자가 아니면 운반 경로의 강탈 구역에서 1회 `자기장 강탈` 사용

계약 포기: `-0.25B`. 도주 또는 강탈 뒤 실패: 총 `-0.5B`. 성공: 보증금 반환 후 계약금 지급.

## 11. 자기장 강탈

계약자가 지정 운반자가 아닐 때만 사용할 수 있다.

- 배터리 운반 경로 중간의 표시된 강탈 구역에서만 가능
- 운반자 4m 안에서 1초간 조준 유지
- 성공하면 배터리 잠금이 풀려 계약자에게 끌려옴
- 즉시 계약자 윤곽, 배터리 현상금과 탈출정 위치 공개
- 운반자는 1초 뒤 행동 회복
- 조준 중 밀치기를 맞거나 시야가 끊기면 실패하며 한 번만 재시도 가능

강탈은 숨은 살인이 아니라 모두가 보는 공개 배신 시작 동작이다. 계약자가 운반 역할을 못 얻어도 계약을 실행할 수 있지만 성공이 보장되지는 않는다.

## 12. 운반자의 방향 선택

황금 배터리를 든 상태에서 별도 배신 버튼은 없다.

- 공동 소켓 방향 안전 통로: 상태 변화 없음
- 탈출정 방향 경계선을 3초 연속 통과: 도주 선언

경계선은 바닥 조명과 경고음으로 명확히 표시한다. 도주 선언 순간 자기 잠금이 풀리고 다음이 공개된다.

- 현재 소지자 윤곽
- 활성 탈출정 위치
- 기본 또는 계약 현상금
- 감소하기 시작한 공동 발진 준비 게이지

## 13. 황금 배터리 손바뀜

도주 선언 또는 자기장 강탈 뒤에는 다음 규칙을 사용한다.

- 방향 고정 돌진에 맞으면 배터리 드롭
- 벽·문·물체에 튕겨 굴러감
- 단독 탈출 성공 이력이 없는 플레이어는 누구나 주울 수 있음
- 성공 이력이 있는 플레이어도 주울 수 있지만 공동 소켓만 활성화
- 마지막 버튼을 5초 유지해야 단독 탈출
- 공동 소켓에 잠기면 다시 꺼낼 수 없음

새 소지자는 배터리를 주운 뒤 3초 동안 원자로와 탈출정 양쪽 경로가 강조된다.

- 원자로 쪽으로 이동: 회수 상태
- 탈출정 쪽으로 3초 이동: 새 도망자 확정

새 도망자가 확정되면 이전 도망자는 실패 `-0.5B`를 받는다.

## 14. 현상금 승계

계약 없는 기본 배터리의 단독 탈출 보상은 `2B`다.

계약자가 도주 또는 강탈을 시작하면 계약 현상금이 배터리에 붙는다.

- 2.5B 계약 → 새 소지자 `2B`
- 3B 계약 → 새 소지자 `2.5B`
- 3.5B 계약 → 새 소지자 `3B`

새 소지자는 원계약자보다 0.5B 적게 받고 가속·문 잠금도 없다. 대신 탈취 위치에 따라 탈출정과 가까울 수 있다.

## 15. 종료 분기

### 공동 발진

- 배터리 공동 소켓 잠금 후 세 설비 3-of-3 유지 8초
- 도주하지 않은 플레이어 `+1B`
- 도주 회수에 기여한 충성 플레이어 각각 `+0.15B`
- 기여: 도망자 밀치기, 배터리 회수 운반, 비어 있는 설비 유지 중 하나
- 실패한 도망자는 `-0.5B`, 공동 보상 없음

### 단독 탈출

- 마지막 버튼 완료 시 소지자가 기본 또는 계약 현상금 획득
- 남은 세 명은 20초 동안 2-of-3 비상 발진
- 비상 발진 성공: 각각 `+0.5B`
- 실패: 보상 없음

### 전체 시간 초과

- 전원 `-0.25B`
- 현재 도망자는 총 `-0.75B`

단독 탈출은 남은 플레이어를 즉시 제거하지 않는다.

## 16. 충성 플레이어의 재미

- 물리 협동 수리와 숙련 보상
- 지정된 설비를 유지하며 운반자의 움직임 관찰
- 자기장 강탈 조준을 밀쳐 끊기
- 설비를 버리고 추격할지 즉시 결정
- 배터리 탈취 후 반환과 현상금 중 선택
- 도주자 이탈 뒤 3인의 비상 발진

누구나 한 번은 지정 운반자가 되고, 계약이 없어도 기본 단독 탈출 선택을 받는다.

## 17. 맵과 몬스터

고정 방 골격에서 고장 설비, 잠긴 문, 제작기, 공동 소켓, 강탈 구역, 탈출정과 느린 순찰 몬스터 위치를 바꾼다.

몬스터는 큰 소음과 배터리 운반자를 추적한다. 운반자는 몬스터를 추격자에게 유인할 수 있고 충성자는 길을 열 수 있다. 직접 전투와 영구 사망은 없다.

```text
→ 갈림길 방향 선택 또는 공개 자기장 강탈
→ 설비를 지킬지 추격할지 결정
→ 밀치기·드롭·현상금 전염

→ 공동 또는 단독·비상 발진
→ 즉시 정산
```

## 19. 차별성 검증 게이트

다음 질문에 하나라도 실패하면 규칙을 추가하지 않고 핵심 구조를 다시 검토한다.

1. 영상만 보고 배터리를 뺏은 사람이 새 배신자가 됐음을 이해하는가?
2. 플레이어가 핵심을 범인 찾기가 아니라 방향 선택과 소유권 이전으로 설명하는가?
3. 계약이 한 번도 없어도 네 명 모두 한 번씩 실제 배신 선택을 받는가?
4. 마지막 장면이 회의가 아니라 밀치기와 버튼 유지인가?
5. 충성 플레이어가 설비 유지·강탈 저지·회수에서 능동적으로 기여하는가?

## 20. 초기 검증 범위

- 친구 4명, Discord 음성
- 4스테이지, 운반자 1회씩 순환
- 맵 1개와 위치 조합 3개
- 수리 4종, 코미디 사고 2종
- 계약금 3단계, 보증금과 자기장 강탈
- 배터리 자기 잠금·드롭·튕김·현상금 승계
- 3-of-3 공동 발진, 2-of-3 비상 발진
- 방향 고정 돌진과 마지막 버튼

작업 도장, 후보 패널, 보정 키, 펄스 경제, 회의, 투표, 자체 음성, 공개 매칭과 영구 성장은 제외한다.

## 21. 검증 기준

- 10초 무음 영상에서 소유권과 배신 기회 이전 이해율 `70% 이상`
- 설명 후 어몽어스형 추리 게임 분류 `25% 미만`
- 각 플레이어가 운반 차례에 실제로 선택을 고민했다는 응답 `75% 이상`
- 계약 없는 운반 차례 도주 시도율 `15~40%`
- 계약 수락률 `25~55%`
- 자기장 강탈 성공률 `30~60%`
- 배터리 소유권 변경 평균 `0.8~2회`
- 배터리 획득 후 반환·도주 고민 응답 `60% 이상`
- 설비 담당의 감시·추격 이탈 스테이지 `30~70%`
- 충성·도주 만족도 차이 1점 이내
- 마지막 버튼 니어미스 `30% 이상`
- 핵심 설명 60초 이내
- 4스테이지 재플레이 의향 `60% 이상`

## 22. 폐기 기준

12회의 4인 세션 뒤 다음 중 하나가 지속되면 구조를 폐기한다.

- 지정 운반자가 거의 항상 공동 소켓 또는 거의 항상 탈출정 선택
- 자기 잠금 때문에 추격 전 긴장이 사라짐
- 자기장 강탈이 확정 성공 또는 사실상 불가능
- 설비 담당이 반복적으로 재미를 희생
- 배터리 손바뀜이 없거나 끝없이 반복
- 이미 단독 탈출한 플레이어의 이후 운반 차례가 무의미함
- 단독 탈출이 전략적 욕심이 아니라 트롤로 인식
- 플레이어가 핵심을 배신자 추리로 설명

## 23. 유사작 대조

- LOCKDOWN Protocol, Project Winter와 First Class Trouble은 고정된 적대 진영과 정체 은폐가 핵심이다. v0.16은 모든 플레이어가 순환 운반자이며 공개 행동 이후에만 적대가 시작된다.
- 일반적인 깃발·보물 운반 게임은 팀 소유권이 고정된다. v0.16은 같은 배터리를 회수한 추격자가 즉시 개인 탈출 보상으로 전환할 수 있다는 점을 핵심으로 둔다.
- 검색 가능한 비교 사례에서 `협동 발진 필수 물건`, `순환 운반`, `회수자의 즉시 배신`, `마지막 버튼 유지`가 함께 결합된 직접 선례는 확인하지 못했다.

## 24. 현재 상태

현재 Unity 프로토타입은 v0.2 규칙을 구현한다. v0.16은 문서 검증 기준안이며 실제 4인 프로토타입 전에는 재미 검증을 통과한 것으로 간주하지 않는다.
