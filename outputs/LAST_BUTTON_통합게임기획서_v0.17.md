# LAST BUTTON 통합 게임 기획서 v0.17

작성일: 2026-07-12  
문서 상태: 차별성 반복 검증 최종 기준안  
대상: 마이크를 사용하는 친구 4명  
플레이 시간: 스테이지당 4~6분, 총 4스테이지

## 1. 한 줄 콘셉트

공동 발진에 필요한 황금 배터리를 한 명이 운반해 원자로에 넣으면 모두 보상받지만, 개인 탈출정으로 방향을 틀면 혼자 큰돈을 노릴 수 있고 추격자가 배터리를 빼앗는 순간 그 욕심까지 물려받는 4인 협동 액션 게임.

## 2. 장르 선언

LAST BUTTON은 사회적 추리 게임이 아니다.

```text
누가 배신자인지 맞힌다 X
공동 물건을 든 사람이 지금 어디로 가는지 본다 O
```

- 고정 임포스터 없음
- 회의·투표·처형·후보 압축 없음
- 정체 추리가 진행 조건이 아님
- 황금 배터리는 매 스테이지 반드시 등장
- 배신 선언 즉시 소지자가 공개됨
- 배터리 소유권과 배신 기회가 함께 이전됨
- 클라이맥스는 마지막 버튼 앞 물리 추격임

## 3. 대표 장면

```text
세 명은 발진 설비를 예열하고 한 명은 황금 배터리를 운반
→ 운반자가 갈림길에서 개인 탈출정 방향으로 틂
→ 경보가 울리고 설비 담당이 자리를 버릴지 결정
→ 도망자가 돌진에 맞아 배터리를 떨어뜨림
→ 회수자가 원자로 대신 탈출정으로 다시 달림
→ 마지막 버튼 5초 동안 배터리 소유권이 또 바뀜
```

10초 무음 영상에서 `잡던 사람이 새 도망자가 됐다`가 이해되어야 한다.

## 4. 세션과 스포트라이트 순환

한 세션은 4스테이지다. 각 스테이지에는 황금 배터리 최초 운반자 한 명이 필요하다.

게임은 `스포트라이트 경험`이 없는 플레이어를 다음 최초 운반자로 우선 지정한다.

스포트라이트 경험으로 인정하는 상황:

- 최초 운반자로 공동 소켓·탈출정 선택을 받음
- 자기장 강탈로 공개 도주를 시작함
- 떨어진 배터리를 주워 반환·도주 양쪽이 열린 상태를 경험함

아직 경험하지 못한 플레이어가 있으면 그중 무작위로 지정한다. 네 명 모두 경험한 뒤에는 최초 운반 횟수가 가장 적은 사람을 우선한다.

이미 단독 탈출한 플레이어는 이후 포드가 잠기며, 스포트라이트 미경험자보다 최초 운반 우선순위가 낮다. 핵심 선택을 못 해본 사람이 남지 않도록 한다.

## 5. 개인 지분

- 0~1.99B: C급
- 2~3.49B: B급
- 3.5~4.99B: A급
- 5B 이상: S급

단일 1등과 탈락은 없다.

- 공동 발진 성공: 전원 `+1B`
- 수리 개인 성과: 스테이지당 최대 `+0.5B`
- 기본 단독 탈출: `+2B`
- 블랙 계약 단독 탈출: `+2.5B / +3B / +3.5B`
- 공개 도주 또는 강탈 뒤 실패: `-0.5B`

공동 성공 4회와 평균 개인 성과 `0.25B` 이상으로 충성 플레이도 S급에 도달한다. 단독 탈출 보상은 플레이어당 세션 1회만 받을 수 있다.

## 6. 다음 스테이지 보급

누적 지분 최고 기록이 처음 정수 단위를 넘을 때마다 일회용 보급 카드 한 장을 받는다. 다음 스테이지에 한 장만 장착하며 사용해도 최종 지분은 줄지 않는다.

- 충격 흡수화: 밀치기 또는 미끄러짐 1회 감소
- 예비 퓨즈: 비어 있는 설비를 4초 임시 유지
- 문 우회키: 잠긴 문 하나를 짧게 개방
- 호스 클램프: 냉각수 사고 1회 봉합

## 7. 한 스테이지 흐름

1. 고장·통로·최초 운반자 공개
2. 약 3분 협동 수리와 물리 사고
3. 중간에 비공개 블랙 계약 제안
4. 수리 완료 후 세 명은 발진 설비, 운반자는 배터리 제작기 배치
5. 네 위치 3초 유지 후 역할 잠금과 30초 최종 단계 시작
6. 세 명은 설비 예열, 한 명은 황금 배터리 운반
7. 도주 선언 또는 계약 강탈 시 공개 추격
8. 공동 소켓·마지막 버튼·3인 비상 발진 중 하나로 종료
9. 즉시 지분 정산

## 8. 협동 수리

- 전력 릴레이: 떨어진 두 방에서 박자를 맞춤
- 냉각수 호스: 호스를 끌고 밸브 조작, 꼬이면 물과 미끄러운 바닥 발생
- 압력 밸브: 숫자를 전달하고 과압 조절, 실패하면 짧은 증기 분출
- 에너지 셀: 무거운 셀을 함께 운반

실패는 10~20초 안에 복구되고 전체 진행도를 초기화하지 않는다. 개인 성과는 도움·정확도·운반량에서 얻으며 한 업무 반복으로 상한을 채우지 못한다.

## 9. 발진 예열과 호위 비용

역할이 잠기는 순간부터 30초 최종 타이머와 세 설비 예열이 동시에 시작된다.

- 원자로·냉각수·통신 담당은 자기 위치를 유지해 예열 게이지를 최대 8초까지 채움
- 자리를 뜨면 1초 뒤 예열이 초당 2초씩 감소
- 자리에서 2초 이상 이탈하면 복귀 후 5초 재부팅을 마쳐야 예열을 다시 채울 수 있음
- 황금 배터리가 공동 소켓에 들어왔을 때 세 게이지가 모두 가득 차면 3초 점화 후 공동 발진
- 부족한 게이지가 있으면 담당자가 돌아와 다시 채워야 함
- 최종 타이머가 끝나면 전체 실패

전원이 운반자를 끝까지 호위하면 세 설비 모두 재부팅이 걸려 30초 안의 공동 발진이 불가능하도록 초기 맵 이동 시간을 맞춘다. 한 명이 짧게 호위하고 일찍 복귀하는 것은 가능하지만, 두 설비만 준비된 상태에서 강탈·도주를 저지해야 한다.

## 10. 배터리 자기 잠금

역할 잠금 뒤 황금 배터리는 최초 운반자와 자기 결합한다.

- 도주 선언 전 일반 돌진으로는 배터리가 떨어지지 않음
- 운반자의 몸은 밀리므로 길막과 코미디는 가능
- 안전 통로에서 같은 플레이어가 2초 안에 연속 두 번 밀면 공격자 설비의 예열이 즉시 3초 감소
- 탈출 경계 안, 도주 선언 뒤, 자기장 강탈 중에는 이 안전 페널티가 없음

무비용 선제 밀치기로 운반자를 공동 소켓까지 강제하는 전략을 억제한다.

## 11. 블랙 계약

- 아직 계약을 수락하지 않았고 단독 탈출에 성공하지 않은 플레이어만 대상
- 무작위 한 명에게 12초 비공개 제안
- 거절하면 다른 한 명에게 마지막 제안
- 계약금 `2.5B / 3B / 3.5B` 중 무작위
- 수락 즉시 보증금 `0.25B` 잠금
- 계약 수락은 플레이어당 세션 1회
- 존재와 수락자는 실행 전까지 비공개

수락자는 `검은 우회권`을 얻는다.

- 제작기와 활성 탈출정 조합을 미리 확인
- 자신이 최초 운반자면 도주 선언 직후 2초 가속과 문 하나 4초 잠금
- 운반자가 아니면 강탈 구역에서 자기장 강탈 1회

포기 시 `-0.25B`, 공개 실행 뒤 실패 시 총 `-0.5B`, 성공 시 보증금 반환 후 계약금을 받는다.

## 12. 자기장 강탈

계약자가 최초 운반자가 아닐 때만 사용할 수 있다.

- 표시된 강탈 구역에서 운반자 4m 안을 1초 조준
- 성공하면 자기 잠금이 풀리고 배터리가 계약자에게 이동
- 즉시 계약자 윤곽, 현상금과 탈출정 공개
- 조준 중 밀치기 또는 시야 단절 시 실패
- 한 번 실패하면 3초 뒤 마지막 한 번만 재시도

강탈은 공개 배신 시작 동작이다. 설비 담당 계약자는 강탈 구역으로 이동하는 동안 자기 예열을 포기한다.

## 13. 방향으로 하는 배신 선언

최초 운반자에게 별도 배신 버튼은 없다.

- 공동 소켓 안전 통로: 상태 변화 없음
- 탈출정 경계선을 3초 연속 통과: 도주 선언

경계선은 바닥 조명과 경고음으로 명확히 표시한다. 선언 순간 자기 잠금이 풀리고 소지자 윤곽, 탈출정, 현상금과 감소 중인 발진 게이지가 공개된다.

## 14. 배터리 손바뀜과 욕심 전염

도주 선언 또는 자기장 강탈 뒤에는 배터리가 완전한 물리 운반물이 된다.

- 방향 고정 돌진에 맞으면 드롭
- 벽·문·물체에 튕겨 굴러감
- 단독 탈출 이력이 없는 플레이어는 누구나 주울 수 있음
- 성공 이력이 있는 플레이어도 주울 수 있지만 공동 소켓 경로만 활성화
- 공동 소켓에 잠기면 다시 꺼낼 수 없음
- 마지막 버튼을 5초 유지해야 단독 탈출

새 소지자는 3초 동안 두 경로를 본다.

- 공동 소켓 방향 이동: 회수 상태
- 탈출정 방향 3초 이동: 새 도망자 확정

새 도망자가 확정되면 이전 도망자는 실패 `-0.5B`를 받는다.

## 15. 현상금 승계

계약 없는 기본 배터리는 단독 탈출 `+2B`다.

계약이 실행된 배터리는 다른 플레이어에게 넘어갈 때 계약금보다 0.5B 낮은 현상금을 가진다.

- 2.5B 계약 → `2B`
- 3B 계약 → `2.5B`
- 3.5B 계약 → `3B`

새 소지자는 가속과 문 잠금을 받지 않지만 탈취 위치가 탈출정에 가까울 수 있다.

## 16. 종료와 보상

### 공동 발진

- 배터리 잠금, 세 설비 완전 예열, 3초 점화
- 도주하지 않은 플레이어 `+1B`
- 도주 회수 기여자 각각 `+0.15B`
- 기여: 도망자 밀치기, 배터리 회수 운반, 비어 있는 설비 유지
- 실패한 도망자 `-0.5B`, 공동 보상 없음

### 단독 탈출

- 마지막 버튼 완료 시 현재 소지자가 현상금 획득
- 남은 세 명은 별도 20초 동안 2-of-3 비상 발진
- 성공 시 각각 `+0.5B`, 실패 시 보상 없음

### 최종 타이머 실패

- 전원 `-0.25B`
- 현재 도망자는 총 `-0.75B`

단독 탈출은 남은 플레이어를 즉시 제거하지 않는다.

## 17. 충성 플레이어의 재미

- 물리 협동 수리와 개인 숙련
- 자기 설비 예열을 지킬지 호위할지 선택
- 자기장 강탈 조준을 밀쳐 끊기
- 공개 도주 뒤 설비를 버리고 추격할지 결정
- 배터리 회수 후 반환과 현상금 중 선택
- 다른 플레이어가 도망친 뒤 3인 비상 발진

계약을 못 받아도 스포트라이트 순환으로 실제 방향 선택을 경험한다.

## 18. 맵과 몬스터

고정 방 골격에서 고장 설비, 잠긴 문, 제작기, 공동 소켓, 강탈 구역, 탈출정과 느린 순찰 몬스터 위치를 바꾼다.

몬스터는 큰 소음과 배터리 운반자를 추적한다. 운반자는 몬스터를 추격자 쪽으로 유인할 수 있고 충성자는 반대로 길을 열 수 있다. 직접 전투와 영구 사망은 없다.

## 19. DDD 리듬

```text
협동 수리 성공
→ 물리 사고와 빠른 복구
→ 지분·다음 장비 진행
→ 손실 있는 비밀 계약
→ 발진 역할 잠금과 30초 카운트다운
→ 설비 예열과 황금 배터리 운반 동시 진행
→ 갈림길 방향 선택 또는 공개 강탈
→ 설비 이탈 결정과 추격
→ 밀치기·드롭·현상금 전염
→ 마지막 버튼 5초 니어미스
→ 공동 또는 단독·비상 발진
→ 즉시 정산
```

## 20. 차별성 검증 게이트

1. 영상만 보고 배터리를 뺏은 사람이 새 도망자가 됐음을 이해하는가?
2. 핵심을 범인 찾기가 아니라 방향 선택과 소유권 이전으로 설명하는가?
3. 계약이 없어도 모든 플레이어가 세션 중 실제 배신 선택을 경험하는가?
4. 설비 예열 때문에 전원 호위가 고정 정답이 되지 않는가?
5. 마지막 장면이 회의가 아니라 밀치기와 버튼 유지인가?

하나라도 실패하면 증거·직업 규칙을 추가하지 않고 배터리 운반 구조를 다시 검토한다.

## 21. 초기 검증 범위

- 친구 4명, Discord 음성
- 4스테이지와 스포트라이트 순환
- 맵 1개, 위치 조합 3개
- 수리 4종, 코미디 사고 2종
- 계약금 3단계, 보증금, 자기장 강탈
- 발진 예열, 배터리 잠금·드롭·튕김·현상금 승계
- 3-of-3 공동 발진, 2-of-3 비상 발진
- 방향 고정 돌진과 마지막 버튼

작업 도장, 후보 패널, 보정 키, 펄스 경제, 회의, 투표, 자체 음성, 공개 매칭과 영구 성장은 제외한다.

## 22. 검증 기준

- 10초 무음 영상에서 소유권과 배신 기회 이전 이해율 `70% 이상`
- 설명 후 어몽어스형 추리 게임 분류 `25% 미만`
- 네 명 모두 스포트라이트 선택 경험 `100%`
- 계약 없는 최초 운반자의 도주 시도율 `15~40%`
- 계약 수락률 `25~55%`
- 자기장 강탈 성공률 `30~60%`
- 배터리 소유권 변경 평균 `0.8~2회`
- 획득 후 반환·도주 고민 응답 `60% 이상`
- 설비 이탈 스테이지 `30~70%`
- 전원 호위 스테이지 `30% 미만`
- 충성·도주 만족도 차이 1점 이내
- 마지막 버튼 니어미스 `30% 이상`
- 핵심 설명 60초 이내
- 4스테이지 재플레이 의향 `60% 이상`

## 23. 폐기 기준

12회의 4인 세션 뒤 다음 중 하나가 지속되면 구조를 폐기한다.

- 운반자가 거의 항상 공동 소켓 또는 거의 항상 탈출정 선택
- 전원 호위 또는 전원 설비 유지가 고정 정답
- 자기 잠금으로 배신 전 긴장이 사라짐
- 자기장 강탈이 확정 성공 또는 사실상 불가능
- 설비 담당이 재미를 반복 희생
- 배터리 손바뀜이 없거나 끝없이 반복
- 단독 탈출이 전략적 욕심이 아니라 트롤로 인식
- 플레이어가 핵심을 배신자 추리로 설명

## 24. 유사작 대조

- LOCKDOWN Protocol, Project Winter와 First Class Trouble은 고정 적대 진영과 정체 은폐가 핵심이다. v0.17은 공개 행동 전까지 적대 진영 자체가 없다.
- 깃발·보물 운반 게임은 소유자의 팀 목표가 고정되는 경우가 일반적이다. v0.17은 회수자가 같은 물건으로 즉시 개인 탈출을 선택할 수 있다.
- 검색 가능한 비교 사례에서 `협동 발진 필수 물건`, `소유권과 배신 기회의 동시 이전`, `설비 예열과 추격의 교환`, `버튼 유지`가 함께 결합된 직접 선례는 확인하지 못했다.

## 25. 현재 상태

현재 Unity 프로토타입은 v0.2 규칙을 구현한다. v0.17은 문서 검증 기준안이며 실제 4인 플레이테스트 전에는 재미 검증 통과로 간주하지 않는다.
