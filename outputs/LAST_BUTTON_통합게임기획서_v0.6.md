# LAST BUTTON — 통합 게임 기획서 v0.6

작성일: 2026-07-12  
문서 상태: 반복 검증안  
대상: 마이크를 사용하는 친구 4명  
플레이 시간: 5–7분 × 3스테이지

## 1. 핵심 정의

네 명의 계약직 정비공이 고장 난 함선을 함께 살리면서 개인 성과 지분을 경쟁한다. 스테이지 도중 누구든 비공개 블랙 계약을 제안받을 수 있고, 이를 수락한 사람은 의무 협동 작업 중 동료에게 잘못된 작업 지시를 내려 개인 포드 전력을 빼돌린다.

3스테이지 종료 후 지분 1위가 승진한다. 함선 생존은 전원에게 필요한 공통 조건이고 블랙 계약은 순위를 뒤집는 고위험 선택이다.

| 동시 선착순 수락 | 무작위 한 명에게 순차 비공개 제안 |

| 계약 성공 후 남은 판이 불명확 | 마지막 버튼 성공 즉시 스테이지 종료 및 정산 |
| 경쟁자가 서로의 개인 업무를 방해 | 개인 성과는 비독점·상한제·공동 성공 배수로 계산 |

## 3. 세션의 승리

- 스테이지마다 개인 성과 지분과 결과 보너스를 획득함
- 3스테이지 종료 후 총지분 1위가 승진함
- 함선이 실패하면 전원이 해당 스테이지 결과 보너스를 잃음
- 개인 성과 지분은 함선 실패 시에도 절반만 보존함
- 블랙 계약 성공은 개인 순위를 크게 바꿈
- 블랙 계약이 없어도 협동 숙련과 합법 업무로 최종 1위가 가능함

## 4. 개인 성과의 비경쟁화

개인 지분은 다른 사람의 기회를 빼앗는 마지막 타격 방식으로 지급하지 않음.

### 지급 원칙

- 협동 작업은 참여자 모두에게 지분 지급
- 구조는 구조자와 구조받은 사람 모두에게 협동 지분 지급
- 화물은 운반 거리보다 무손상 공동 완료를 평가함
- 수리 유형별 지분에는 스테이지 상한이 있음
- 같은 행동 반복 파밍은 지분을 추가 지급하지 않음
- 공동 발진 성공 시 개인 성과에 팀 성공 배수를 적용함
- 고의 밀치기와 작업 중단은 발진 경보 전에는 지분을 차감함

한 플레이어가 다른 플레이어의 완벽 수리 기회를 빼앗거나 마지막 입력만 가로채는 구조를 사용하지 않음.

## 5. 합법 개인 업무

모든 플레이어는 매 스테이지 공동 목표와 충돌하지 않는 합법 업무 하나를 받음.

예시:

- 서로 다른 두 수리 작업에 보조자로 참여
- 손상 없이 화물 운반 완료
- 두 종류의 설비 수리 완료
- 동료와 함께 감전 사고 없이 회로 복구
- 최종 발진 동안 서로 다른 두 설비 안정화

합법 업무는 희귀 자원을 독점하거나 수리를 지연해야 달성되는 조건을 사용하지 않음.

플레이어는 보안 단말기에서 자신의 합법 업무를 공개할 수 있음. 공개해도 블랙 계약 보유 여부는 증명되지 않으며 합법 업무 보너스의 25%를 잃음. 공개는 동선 설명에 신뢰를 더하지만 완전한 결백 증명은 아님.

## 6. 블랙 계약 제안

### 대상

- 순위와 무관하게 네 명 모두 대상이 될 수 있음
- 이전 스테이지 계약자는 다음 스테이지 제안 확률만 낮아지며 제외되지는 않음
- 한 명에게 비공개로 12초간 제안함
- 거절하면 무작위 다른 플레이어에게 한 번 더 제안함
- 두 명 모두 거절하면 계약자 없이 진행함
- 스테이지당 계약자는 최대 한 명임

누가 제안을 받았는지, 몇 번째 제안에서 수락했는지와 제안 보상은 공개하지 않음.

### 가변 계약 조건

계약 제안은 다음 중 하나의 성공 보상을 가짐.

- 보수적 계약: 성공 `2.5B`, 실패 `-0.5B`
- 표준 계약: 성공 `3B`, 실패 `-0.5B`
- 고액 계약: 성공 `3.5B`, 실패 `-0.5B`

제안받은 플레이어는 현재 순위, 남은 스테이지와 제시된 배수를 보고 수락 여부를 결정함.

### 존재 경보

계약 수락 15–25초 후 다음 방송이 발생함.

> 불법 전력 계약이 활성화되었습니다.

계약자의 존재만 공개하고 후보를 좁히는 순위 정보는 제공하지 않음.

## 7. 의무 협동 기만 작업

모든 스테이지에는 발진에 필요한 `위상 보정` 작업이 한 번 존재함. 계약자 여부와 관계없이 반드시 수행해야 하므로 특정 플레이어의 요청을 거부하는 것으로 계약을 봉쇄할 수 없음.

### 역할

- 판독자: 중앙 모니터에서 올바른 기호와 순서를 확인함
- 조작자: 다른 방에서 판독자가 말한 기호대로 스위치를 입력함

### 정상 플레이

판독자가 올바른 기호를 전달하면 공동 발진 회로가 활성화됨.

### 블랙 계약 플레이

계약자가 판독자일 때만 화면에 비밀 포드 회로가 함께 표시됨. 계약 성공 준비를 위해서는 조작자에게 정상 답이 아닌 포드 회로의 기호를 전달해야 함.

- 계약자가 조작자일 때는 비밀 전용을 실행할 수 없음
- 계약자는 판독자 역할을 확보해야 함
- 조작자는 입력 전에 판독자의 음성 또는 핑 지시를 받아야 함
- 포드 회로 입력도 당장은 정상 성공처럼 보임
- 계약자는 실제로 틀린 정보를 전달해야 하므로 단순 메뉴 선택만으로 진행할 수 없음

### 역할 배정

위상 보정 작업이 열리면 네 명 중 두 명이 자발적으로 판독자와 조작자를 맡음. 동일 플레이어가 연속 스테이지에서 같은 역할을 맡으면 해당 역할의 개인 지분이 감소함.

## 8. 지연된 이상과 정보

위상 보정 완료 20–35초 후 포드 전용이 있었다면 다음 현상이 발생함.

- 공동 전력 순간 감소
- 세 설비 중 하나의 출력 불안정
- 비인가 전력 경보
- 위상 보정이 정상 회로가 아니었다는 사실

시스템은 판독자와 조작자의 이름을 로그에 남기지 않음. 플레이어는 누가 어떤 역할을 맡았는지 자신의 목격과 대화로 기억해야 함.

정상적인 재난도 비슷한 전력 불안정을 만들 수 있지만 `불법 전력 계약 활성화` 방송은 실제 계약이 수락된 판에만 발생함.

## 9. 충성 플레이어의 대응

계약자 역할을 확정하거나 특정 플레이어를 감금하는 기능은 없음. 충성 플레이어는 제한된 설비 대응을 선택함.

### 변조 봉인

- 세 위상 보정 콘솔 중 하나에 스테이지당 한 번 설치함
- 봉인된 콘솔에서 포드 회로가 입력되면 즉시 해당 구역 경보가 울림
- 플레이어 이름은 공개하지 않음

### 포드 회로 차단

- 불법 전력 경보 후 한 명이 12초 작업하면 저장된 포드 전력을 제거함
- 작업 중 공동 발진 수리에는 참여할 수 없음
- 계약자는 차단 전에 키카드 실행 단계로 넘어가야 함

### 통로 봉쇄

- 포드로 향하는 두 통로 중 하나를 20초간 잠금
- 특정 플레이어가 아니라 모든 플레이어에게 적용됨
- 잘못된 통로를 잠그면 계약자는 반대편으로 이동함

### 키카드 회수

- 실행 단계의 물리 키카드는 누구나 빼앗고 보안함에 반환할 수 있음
- 반환에는 5초가 필요하며 밀치기로 중단 가능함

## 10. 공동 최종 발진

최종 30초 동안 세 설비가 활성화됨.

- 원자로 안정기
- 항법 계산기
- 화물 고정 장치

세 설비 중 두 개가 작동 중이면 발진 게이지가 유지되고, 세 개가 모두 작동하면 더 빠르게 증가함. 한 개 이하만 작동하면 게이지가 감소함.

### 계약자가 없는 판

- 두 명이 핵심 설비 유지
- 한 명이 돌발 고장 처리
- 한 명이 화물 또는 보너스 설비 담당

### 계약자가 실행하는 판

- 계약자 한 명이 포드로 이동함
- 충성 플레이어 세 명 중 두 명이 발진 설비 유지
- 한 명이 포드 차단, 통로 봉쇄 또는 추격을 선택함
- 설비 담당도 상황에 따라 자리를 바꿀 수 있음

계약자 이탈만으로 공동 발진이 자동 실패하지 않음.

## 11. 마지막 버튼과 스테이지 종료

계약자가 포드 전력과 키카드를 확보하면 마지막 버튼을 5초간 유지할 수 있음.

- 돌진은 조준 방향으로 고정되며 빗나갈 수 있음
- 키카드 소지자가 맞으면 키카드를 떨어뜨림
- 통로와 설비 상태가 추격 경로를 바꿈
- 버튼 성공 즉시 개인 탈출 성공으로 스테이지 종료
- 계약자는 제안된 성공 보상을 획득
- 다른 플레이어는 각각 `0.75B`와 자신의 개인 성과 지분을 획득
- 버튼 실패 또는 공동 발진 선행 성공 시 계약 실패 `-0.5B`
- 공동 발진 게이지가 먼저 완료되면 즉시 공동 성공으로 스테이지 종료

개인 탈출 후 남은 플레이를 따로 진행하지 않아 관전 공백을 만들지 않음.

## 12. 코미디 수리

계약자가 없는 판도 재미있어야 하므로 수리는 짧은 협동 코미디로 구성함.

### 공통 원칙

- 설명 5초 이내
- 실행 10–20초
- 성공과 부분 성공 존재
- 실패해도 전체 초기화 없음
- 실수는 1–3초 물리 사고를 만들고 장기 조작 불능을 만들지 않음
- 개인 지분 손실보다 팀 진행 지연과 장면 연출 중심
- 네트워크 판정은 연속 로프 물리보다 제한된 상태와 사전 정의된 충격을 우선함

### 초기 작업 2종

#### 냉각수 호스

- 한 명이 호스 끝을 잡고 다른 한 명이 밸브를 잠금
- 놓치면 호스가 정해진 범위에서 플레이어를 밀치고 바닥을 미끄럽게 함
- 실패해도 밸브 잠금 진행 일부 유지

#### 압력 밸브

- 한 명은 압력 숫자를 보고 다른 한 명은 밸브를 조작함
- 과압 시 짧은 증기 충격과 물건 드롭 발생
- 목표 구간 유지 시간에 따라 정상 또는 완벽 수리 판정

거대 배터리, 크레인과 덕트는 핵심 루프 통과 후 추가함.

## 13. 충성 플레이의 보상 순간

- 합법 업무 완료
- 협동 수리 정상·완벽 판정
- 사고 직후 복구
- 불법 전력 이전에 변조 봉인 적중
- 포드 회로 차단
- 올바른 통로 봉쇄
- 돌진으로 키카드 드롭
- 공동 발진 클러치
- 결과 화면의 실제 계약자 공개
- 지분 순위 상승

충성 플레이의 최고점은 계약자를 맞히는 투표가 아니라 수리 숙련, 설비 판단과 물리적 저지에서 발생함.

## 14. DDD 리듬

```text
작은 수리 성공
→ 개인 지분 상승
→ 합법 업무 진행
→ 계약 유혹 또는 계약 경보
→ 의무 협동 기만
→ 지연된 이상
→ 설비 대응 선택
→ 최종 발진과 포드 경쟁
→ 마지막 버튼 니어미스
→ 계약자 공개와 순위 역전
```

계약을 받지 않은 플레이어에게도 수리 판정, 사고 복구, 설비 대응, 키카드 저지와 공동 발진이라는 반복 보상이 존재함.

- 냉각수 호스와 압력 밸브
- 지연된 전력 이상
- 변조 봉인, 포드 차단과 통로 봉쇄
- 2-of-3 공동 최종 발진
- 키카드, 포드, 돌진과 마지막 버튼
- Discord 음성
- 친구 초대 또는 직접 접속

### 제외

- 6인
- 공개 매치메이킹
- 자체 근접 음성
- 플레이어 대상 스캔과 감금
- 몬스터
- 모듈 맵
- CCTV와 리플레이 영상
- 얼굴 애니메이션
- 전용 서버
- 복수 계약자
- 다수의 블랙 계약 종류

## 16. 검증 지표

### 계약

- 제안 대비 수락률 25–65%
- 수락 대비 성공률 30–45%
- 계약자가 조작자에게 거짓 기호 또는 잘못된 지시를 전달한 판 90% 이상
- 경보 후 특정 순위만 일괄 배제하는 행동 20% 미만
- 결과 공개 전 계약자 확정 추측 성공률 40–75%

### 충성 플레이

- 계약자가 없는 판의 재플레이 의향 60% 이상
- 수리만 반복했다고 응답한 플레이어 25% 미만
- 충성 플레이어가 기억하는 클러치 장면 평균 1개 이상
- 계약자와 충성 플레이 재미 점수 차이 1점 이내

### 전체

- 판당 강한 웃음·비명·언쟁 2회 이상
- 3스테이지 후 재대결 의향 60% 이상
- 최종 승리 조건을 오해한 플레이어 20% 미만
- 고의 비계약 방해가 최적이라고 답한 플레이어 20% 미만

## 17. 폐기 기준

- 위상 보정 파트너가 계약자를 자동 확정함
- 계약 경보 후 특정 플레이어와의 협동을 거부하는 전략이 반복됨
- 계약자가 음성 또는 핑으로 잘못된 정보를 전달하지 않고도 준비를 완료함
- 계약자가 없는 판에서 수리가 재미없음
- 개인 순위 때문에 정상 협동 방해가 반복됨
- 계약 성공이 전략보다 트롤링으로 평가됨
- 계약 수락률이 지속적으로 20% 미만 또는 70% 이상임
- 3스테이지 후 재플레이 의향이 절반 미만임

## 18. 현재 프로토타입과의 차이

현재 Unity 프로토타입은 v0.2 규칙을 구현함. v0.6의 개인 지분, 순차 계약, 위상 보정 기만, 설비 대응, 코미디 수리와 2-of-3 발진은 아직 구현하지 않음.

기획 반복 검증이 끝날 때까지 프로토타입 코드를 변경하지 않음.
