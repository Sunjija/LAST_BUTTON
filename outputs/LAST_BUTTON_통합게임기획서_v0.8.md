# LAST BUTTON — 통합 게임 기획서 v0.8

작성일: 2026-07-12  
문서 상태: 반복 검증안  
대상: 마이크를 사용하는 친구 4명  
플레이 시간: 5–7분 × 3스테이지

## 1. 게임 정의

네 명의 계약직 정비공이 고장 난 함선을 살리고 3스테이지 동안 각자의 고용 할당량을 채운다. 누구든 비공개 블랙 계약을 제안받을 수 있으며, 계약자는 전원이 참여하는 위상 체크섬에서 거짓 기호를 보고해 개인 포드를 준비한다.

다른 플레이어를 이기는 단독 순위는 없음. 각 플레이어는 독립적으로 할당량을 달성하거나 실패함. 공동 성공은 모두에게 가장 안정적인 경로이고 블랙 계약은 부족한 할당량을 한 번에 만회하는 고위험 경로임.

## 2. 핵심 수치

### 세션 할당량

- 고용 유지 기준: `3.5B`
- 공동 성공: `1B`
- 합법 업무와 개인 성과 합계: 스테이지당 최대 `0.25B`
- 블랙 계약 성공: `2.5B`
- 블랙 계약 실패: `-0.5B`
- 다른 플레이어의 블랙 계약 성공: 각자 `0.75B`
- 전체 함선 실패: 공동 보너스 없음, 개인 성과 절반, 모든 플레이어 추가 `-0.25B` 안전 위반 책임

### 정상 충성 경로

```text
공동 성공 3회 = 3B
합법 업무·개인 성과 평균 0.5B 이상
→ 할당량 3.5B 달성
```

### 블랙 계약 피해 경로

```text
공동 성공 2회 = 2B
다른 사람의 계약 성공 보상 = 0.75B
개인 성과 최대 0.75B
→ 최대 3.5B
```

다른 사람의 계약이 한 번 성공해도 세 스테이지에서 개인 성과를 충분히 달성한 플레이어는 할당량을 지킬 수 있음. 개인 성과가 부족한 플레이어는 할당량 실패 위험이 생김.

### 블랙 계약의 기대값

성공 확률을 `p`라고 하면 계약 기대값은 다음과 같음.

```text
2.5p - 0.5(1-p) = 3p - 0.5
```

공동 성공 `1B`와 같아지는 지점은 성공률 50%임. 계약은 평균 기대값이 높은 선택이 아니라, 이미 할당량 달성이 어려운 플레이어가 분산을 감수하는 역전 선택임.

## 3. 제안과 후보 정보

- 순위와 무관하게 모든 플레이어가 제안 대상임
- 무작위 한 명에게 12초 제안
- 거절 시 다른 무작위 한 명에게 마지막 제안
- 모두 거절하면 계약자 없는 판
- 계약 수락 15–25초 후 존재 경보
- 제안을 받은 사람, 거절한 사람과 계약 조건은 비공개
- 다른 플레이어의 정확한 할당량도 공개하지 않음

계약 경보만으로 특정 순위, 역할 또는 플레이어를 후보에서 제외할 수 없음.

## 4. 합법 업무와 성과

- 합법 업무는 공동 목표와 같은 방향의 비독점 조건만 사용함
- 협동 작업 참여자는 모두 같은 기본 성과를 받음
- 완벽 판정 보너스는 참여자 모두에게 지급함
- 유형별 성과에는 스테이지 상한이 있음
- 반복 파밍, 마지막 입력 가로채기와 희귀 자원 독점으로 지분을 얻을 수 없음
- 합법 업무 공개는 가능하지만 블랙 계약 보유 여부를 증명하지 않음
- 합법 업무 공개 시 해당 업무 보너스 25% 감소

한 플레이어의 성과를 낮춰도 자신의 할당량은 증가하지 않음. 함선 전체 실패는 자신에게도 안전 위반 책임 `-0.25B`를 부과함.

## 5. 전원 위상 체크섬

공동 발진 전에 네 명 모두 수행해야 하는 필수 작업임.

### 배정

- 두 명은 청색 회로, 두 명은 적색 회로에 자동 배정됨
- 플레이어마다 개인 패널에 정상 기호 하나가 표시됨
- 계약자 패널에는 정상 기호와 블랙 계약 기호가 함께 표시됨

### 입력

- 각 플레이어는 자신에게 보인 기호를 음성 또는 핑으로 보고함
- 각자 보고한 기호를 자신의 패널에 입력함
- 비계약자가 다른 기호를 입력하면 즉시 일반 오류가 발생함
- 계약자가 블랙 계약 기호를 입력하면 정상 성공처럼 처리됨
- 계약자는 대체 기호를 정상 기호라고 거짓 보고해야 함
- 어느 회로에 배정되어도 계약 실행 가능

전원 참여 작업이라 특정 역할 배정과 협조 거부로 계약을 봉쇄할 수 없음.

## 6. 대칭형 상충 진단

블랙 계약 기호 입력 20–35초 후 전력 이상이 발생함.

각 회로의 두 플레이어는 서로 다른 비공개 진단 결과를 받음.

### 실제 오염 회로

- 계약자: `회로 정상`이라는 위조 결과
- 같은 회로의 충성 플레이어: `회로 오염`이라는 실제 결과

### 정상 회로

- 무작위 충성 플레이어 한 명: `회로 오염`이라는 센서 오작동
- 같은 회로의 다른 충성 플레이어: `회로 정상`이라는 실제 결과

따라서 양쪽 회로에서 정확히 한 명은 `정상`, 한 명은 `오염`을 주장하게 됨.

```text
청색 A: 정상 주장
청색 B: 오염 주장
적색 C: 정상 주장
적색 D: 오염 주장
```

시스템은 한 회로가 실제 오염됐고 다른 회로에는 센서 오작동이 하나 있다는 사실만 알려줌. 수치상 더 높은 확률과 로그 교집합은 존재하지 않음.

계약자는 자신의 회로가 정상이라고 주장하거나 상대 회로가 오염됐다고 설득해야 함. 정상 회로의 오작동을 받은 충성 플레이어도 자신이 본 정보를 사실대로 주장하므로 양쪽에 진짜 충돌이 발생함.

## 7. 회로 재보정

- 팀은 청색 또는 적색 중 하나만 선택 가능
- 두 명이 12초 작업
- 실제 오염 회로 선택: 포드 전력 제거, 계약 실패
- 정상 회로 선택: 포드 전력 유지, 12초 손실
- 결과는 포드 전력 유지 여부만 공개
- 계약자 신원은 결과 화면까지 공개되지 않음

두 회로의 정보 구조가 대칭이므로 시스템 수치만으로 고정 정답을 선택할 수 없음.

## 8. 키카드와 물리 대응

- 계약 준비 성공 시 계약자에게만 키카드 보관 위치 표시
- 키카드를 꺼내면 모두 볼 수 있는 물리 오브젝트가 됨
- 누구나 돌진으로 떨어뜨리고 회수 가능
- 키카드 반환은 5초 유지
- 회로 재보정 성공 시 키카드가 있어도 포드 사용 불가
- 포드 전력이 유지되면 계약자는 마지막 버튼으로 이동함

돌진은 방향 고정 스킬샷이며 빗나가거나 벽에 충돌할 수 있음.

## 9. 2-of-3 공동 최종 발진

최종 30초 동안 원자로, 항법, 화물 고정 설비가 활성화됨.

- 두 설비 작동: 정상 진행
- 세 설비 작동: 빠른 진행
- 한 설비 이하: 게이지 감소
- 한 번 조작한 설비는 6초간 자동 유지
- 한 플레이어가 여러 설비를 순환 가능

계약자 실행 시 충성 플레이어 두 명이 발진을 유지하고 한 명이 회로 재보정, 키카드 회수 또는 추격을 담당할 수 있음.

## 10. 스테이지 종료

### 공동 성공

- 공동 발진 게이지가 먼저 완료됨
- 각자 `1B`와 성과 지분 획득
- 수락한 계약자는 `-0.5B`

### 계약 성공

- 계약자가 마지막 버튼을 5초 유지함
- 즉시 스테이지 종료
- 계약자 `2.5B`와 성과 지분
- 다른 플레이어 각각 `0.75B`와 성과 지분

### 전체 실패

- 어느 발진도 성공하지 못함
- 공동 보너스 없음
- 성과 지분 절반
- 전원 안전 위반 `-0.25B`
- 계약 수락자는 추가 `-0.5B`

성공과 실패 후 전원이 다음 스테이지에 참가하며 관전 공백은 없음.

## 11. 충성 플레이와 코미디 수리

### 냉각수 호스

- 한 명이 호스를 잡고 한 명이 밸브를 잠금
- 놓치면 제한 범위 물줄기와 미끄러운 바닥 발생
- 진행 일부 유지
- 정상·완벽 판정은 참여자 모두에게 지급

### 압력 밸브

- 한 명은 숫자 확인, 한 명은 밸브 조작
- 과압 시 1–3초 증기 충격과 물건 드롭
- 목표 구간 유지로 정상·완벽 판정

### 최종 발진

- 설비 순환
- 돌발 고장
- 추가 화물 확보
- 계약 실행 시 발진 유지와 추격 분담

수리 실패는 개인 점수를 직접 빼앗기보다 팀 진행을 지연함. 같은 사고의 반복 피로는 계약자 없는 판을 별도로 테스트해 판단함.

## 12. 할당량 달성 이후

- 할당량 초과 지분은 세션 종료 후 외형 재화로 변환됨
- 각 스테이지 전체 실패는 이미 확보한 할당량에서도 `-0.25B`를 차감함
- 세션 종료 전까지 고용 유지 상태는 확정되지 않음
- 마지막 스테이지에도 공동 성공과 개인 성과를 지킬 이유가 남음

영구적인 이동, 수리와 밀치기 능력 강화는 없음.

## 13. DDD 리듬

```text
코미디 수리 성공
→ 할당량 상승
→ 합법 업무 보너스
→ 계약 유혹 또는 계약 존재 경보
→ 전원 체크섬과 거짓 기호
→ 양쪽 회로의 상충 증언
→ 한 번뿐인 재보정 선택
→ 키카드 추격과 공동 발진
→ 마지막 버튼 니어미스
→ 계약자 공개와 할당량 정산
```

## 15. 적대 검증 지표

- 체크섬과 진단만으로 계약자가 자동 확정되지 않음
- 두 회로 중 한쪽이 시스템적으로 항상 우월한 선택이 아님
- 계약자가 거짓 기호와 거짓 진단 주장을 사용한 판 90% 이상
- 계약 제안 수락률 20–65%
- 계약 성공률 30–50%
- 팀 실패 고의 유도가 개인적으로 유리하다고 답한 플레이어 10% 미만
- 계약자가 없는 판 재플레이 의향 60% 이상
- 충성과 계약 역할 재미 차이 1점 이내
- 3스테이지 재대결 의향 60% 이상

## 16. 폐기 기준

- 대칭 진단이 추리가 아니라 완전한 동전 던지기로 느껴짐
- 계약자가 말의 설득력과 무관하게 50% 확률로 차단됨
- 체크섬 참여 거부 메타가 발생함
- 할당량 수치가 계약을 항상 거절하거나 항상 수락하게 만듦
- 할당량 달성자가 이후 팀을 방해함
- 계약 없는 판 수리가 재미없음
- 계약 성공이 트롤링으로 평가됨
- 3스테이지 후 재플레이 의향이 절반 미만임
